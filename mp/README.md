# uni-app-ts-template

> uniapp + vite + vue3 + typescript

## use it: clone

```
npx degit zhuo-tech/uniapp-ts-tpl myproject

cd myproject
git init
git remote add origin [your-own-project-repo-url]
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
